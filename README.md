# CS2 Skins Marketplace

## Project Description

The CS2 Skins Marketplace is an online platform for buying, selling, and trading virtual skins for Counter-Strike 2 (CS2). The marketplace allows users to showcase their skins, find great deals, and make secure transactions within a community of gamers.

## Features
- User-friendly interface for easy navigation
- Secure payment gateway for safe transactions
- Profile management for users to track their inventory and purchase history
- Skin details and pricing information
- Robust search and filtering options

## Setup Instructions

To set up the CS2 Skins Marketplace locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/AjAZZZZ613/case-battle.git
   cd case-battle
   ```
2. **Install dependencies** (Node.js and npm are required)
   ```bash
   npm install
   ```
3. **Set up environment variables**
   - Create a `.env` file in the root directory and add your configuration settings.
   
4. **Start the application**
   ```bash
   npm start
   ```
5. **Open your browser and navigate to** `http://localhost:3000`
